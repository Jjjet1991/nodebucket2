About Us

Nodebucket is an application designed to allow users to track and complete user-generated tasks. Tasks can be added, then moved from To-Do column to Completed column and can be deleted by the user. Users can log into their own profiles and tasks remain in their perspective columns when they are logged into the MongoDB Database.

Nodebucket uses cookie services and requires users to sign in using pre-determined user IDâ€™s. Each user is able to create their own tasks and move them through different stages. When a user is done within their profile they can sign out and will be re-directed back to the homepage. Within the application is a Home page, Contact page, and About page. The application also includes a Not Found 404 page.

The project will be created using Angular and the database will be created and stored with MongoDB. SoapUI will be used to test product solutions. Angular Material and Bootstrap will be used to style pages. This project is a MEAN project meaning it uses MongoDB, Express, Angular and Node.js.
